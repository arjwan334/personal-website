## HTML Elements Used

- `<!DOCTYPE html>` : Defines the document as an HTML5 document.
- `<html>` : The root element of the web page.
- `<head>` : Contains the page information such as the title and CSS link.
- `<title>` : Displays the page title in the browser tab.
- `<link>` : Connects the HTML file with the CSS file.
- `<body>` : Contains all visible page content.
- `<header>` : Displays the main heading of the website.
- `<nav>` : Creates the navigation menu.
- `<section>` : Organizes the page into different content sections.
- `<h1>` and `<h2>` : Used for headings.
- `<p>` : Displays paragraphs.
- `<ul>` : Creates an unordered list.
- `<li>` : Represents each item in the list.
- `<a>` : Creates navigation links.
- `<footer>` : Displays the footer information.

## CSS Properties Used

- `background` : Adds background colors and gradients.
- `color` : Changes text color.
- `font-family` : Sets the font style.
- `font-size` : Changes text size.
- `margin` : Adds spacing outside elements.
- `padding` : Adds spacing inside elements.
- `border-radius` : Creates rounded corners.
- `box-shadow` : Adds shadow effects.
- `text-align` : Aligns text.
- `transition` : Creates smooth animation effects.
- `transform` : Adds hover movement effects.
